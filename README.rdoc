== Хорошие вопросы

Аналог AskFm на rails 4.2. Позволяет регистрироваться, задавать друг другу вопросы и отвечать на них.

Рассчитано на работу с Heroku.

== Демо

https://goodquestions.herokuapp.com

== Установка

Для разворачивания своей копии клонируйте гит репозиторий (зеленая кнопочка в правом верхнем углу на гитхабе), залейте на свое приложение на Heroku (или создайте новое <tt>heroku create</tt>).
